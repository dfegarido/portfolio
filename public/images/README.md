# Portfolio Images

Place your portfolio images in this directory.

## Required Images

1. **Profile Image**: `profile.jpg` (or `.png`, `.webp`)
   - Used in the About section
   - Recommended size: 800x800px or larger
   - Will be displayed as a square image

2. **Project Images**: Add project-specific images as needed
   - Name them descriptively (e.g., `shareikna.jpg`, `phillymatch.jpg`)
   - Recommended size: 1200x800px or larger
   - Format: JPG, PNG, or WebP

## How to Get Images from Your Old Portfolio

1. Visit your old portfolio: https://dfegarido.github.io/portfolio/
2. Right-click on images you want to use
3. Select "Save Image As..." or "Inspect Element" to find the image URL
4. Download the images and place them in this `public/images/` directory

## Alternative: Download from GitHub

If your old portfolio is on GitHub:
1. Go to: https://github.com/dfegarido/portfolio
2. Navigate to the images/assets folder
3. Download the images you need
4. Place them in this directory

## Image Optimization

For best performance, consider:
- Using WebP format for smaller file sizes
- Compressing images before adding them
- Using tools like [Squoosh](https://squoosh.app/) or [TinyPNG](https://tinypng.com/)
